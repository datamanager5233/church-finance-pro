🚀 Setup Guide — Google Sheets + Apps Script

Step 1: Create Google Sheet

Create a new Google Sheet with the following tabs (sheets):

  • MemberRegistry — stores all registered members
  • TithePool — auto-generated numbers TBGB0001–TBGB2000
  • GivingRecords — all giving transactions
  • MobileMoneyTx — parsed mobile money transactions
  • Config — church name, MM number, email settings

Step 2: Deploy Google Apps Script

Open Extensions → Apps Script in your sheet. Copy each script from the "Google Scripts" tab and save.

Step 3: Create Google Forms

  • Member Registration Form → links to MemberRegistry tab
  • Tithe Number Request Form → triggers script to assign number

Step 4: Deploy Web App

In Apps Script: Deploy → New Deployment → Web App. Set access to "Anyone". Copy the web app URL.

Step 5: Host Dashboard on GitHub Pages

  • Create a GitHub repo, upload this index.html
  • Go to Settings → Pages → Deploy from main branch
  • Update the SHEET_URL variable in the JS to your deployed Apps Script URL

Step 6: Mobile Money Integration

Use SMS Forwarder (Android app) on the church phone to forward incoming MM SMS to your Apps Script webhook URL. The script parses the SMS and logs the transaction automatically.
